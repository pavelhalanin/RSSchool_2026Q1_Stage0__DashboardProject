## How to run the application

- For Development
    1. Download VS Code
    1. Download Git
    1. Download NodeJS
    1. Clone repo
        ```
        git clone https://github.com/pavelhalanin/Rolling-Scopes-School-stage-0-dashboard.git
        ```
    1. Open VS Code on cloned folder
    1. Start web site
        ```
        npx serve
        ```
- View Production
    1. Open Browser
    1. Open [website](https://pavelhalanin.github.io/Rolling-Scopes-School-stage-0-dashboard/)
